# Crud de gestión de productos con PHP

## Instalación

```bash

# 1. Clonar el repositorio
git https://github.com/BrayanZipa/prueba_gestion_productos_php.git

# 2. Servidor local
Tener un servidor local instalado e inicializarlo (xampp, Laragon, etc)

# 3. Agregar el proyecto
Agregar el proyecto en la ruta correspondiente según el servidor 

# 4. Ejecutar el script de datos de prueba 
Ejecutar el script productos.sql en su gestor de base de datos

# 5. Configurar archivo conexion.php
Configurar archivo con sus credenciales de base de datos (por defecto localhost)

# 6. Abrir el proyecto
Ingrese en el navegador a su http://localhost/prueba_gestion_productos_php/  
o a la ruta según su estructura de carpetas

```
